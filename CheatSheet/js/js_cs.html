I. Mobile Navigation Menu

  • function myMenuFunction(): Controls the mobile navigation menu behavior.
  • var menuBtn = document.getElementById("myNavMenu");: Gets the menu element by ID.
  • if (menuBtn.className === "nav-menu"): Checks if menu is in its default (likely hidden) state.
  • menuBtn.className += " responsive";: Appends the responsive class to show the menu.
  • else: Otherwise, if already responsive...
  • menuBtn.className = "nav-menu";: Resets the class back to hidden/default.

II. Header Shadow on Scroll

  • window.onscroll = function() { headerShadow() };: Executes headerShadow() when the user scrolls.
  • function headerShadow(): Adjusts the header's shadow and height on scroll.
  • const navHeader = document.getElementById("header");: Retrieves the header element.
  • if (document.body.scrollTop > 50 || document.documentElement.scrollTop > 50): Checks if scrolled more than 50px.
  • If true:
    • navHeader.style.boxShadow = "0 1px 6px rgba(0, 0, 0, 0.1)";: Adds a subtle shadow.
    • navHeader.style.height = "70px";: Reduces header height.
    • navHeader.style.lineHeight = "70px";: Adjusts line height.
  • Else:
    • navHeader.style.boxShadow = "none";: Removes shadow.
    • navHeader.style.height = "90px";: Resets header height.
    • navHeader.style.lineHeight = "90px";: Resets line height.

III. Typing Effect (Typed.js)

  • var typingEffect = new Typed(".typedText", { ... });: Uses Typed.js to animate text.
  • ".typedText": Selector for the element where text is animated.
  • The options inside the object include:
    • strings: ["Designers", "Renovators", "Craftsmen", "Builders", "Experts", "EMR"]: List of words to type out.
    • loop: true: Continuously loops the animation.
    • typeSpeed: 100: Typing speed in milliseconds per character.
    • backSpeed: 80: Backspace speed in milliseconds per character.
    • backDelay: 2000: Delay before backspacing.

IV. Scroll Reveal Animations (ScrollReveal.js)

  • const sr = ScrollReveal({ ... });: Initializes ScrollReveal with default options:
    • origin: 'top': Elements come from the top.
    • distance: '80px': Travel distance during reveal.
    • duration: 2000: Animation duration in milliseconds.
    • reset: true: Re-animates each time elements enter the viewport.
  • sr.reveal('.featured-text-card', {});: Reveals the element with featured-text-card using default settings.
  • sr.reveal('.featured-name', { delay: 100 });: Reveals featured-name with a delay of 100ms.
  • sr.reveal('.project-box', { interval: 200 });: Staggers animation for each project-box by 200ms intervals.
  • const srLeft = ScrollReveal({ origin: 'left', ... });: New instance for left-origin animations.
  • srLeft.reveal('.about-info', { delay: 100 }); and srLeft.reveal('.contact-info', { delay: 100 });: Reveal elements from left.
  • const srRight = ScrollReveal({ origin: 'right', ... });: New instance for right-origin animations.
  • srRight.reveal('.skills-box', { delay: 100 }); and srRight.reveal('.form-control', { delay: 100 });: Reveal elements from right.

V. Change Active Navigation Link

  • const sections = document.querySelectorAll('section[id]');: Selects all sections with an ID.
  • function scrollActive(): Determines which nav link is highlighted based on scroll.
    • const scrollY = window.scrollY;: Gets current scroll position.
    • sections.forEach(current => { ... });: Loops through sections to compute:
      • sectionHeight: Height of the current section.
      • sectionTop = current.offsetTop - 50;: Top offset adjusted by 50px.
      • sectionId = current.getAttribute('id');: Section ID.
      • If scrollY falls within the section bounds, the corresponding nav link gets active-link class; otherwise, it is removed.
  • window.addEventListener('scroll', scrollActive);: Listens for scroll events to update the active navigation.
