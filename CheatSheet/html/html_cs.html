I. Structure

  • <!DOCTYPE html>: Declares HTML5 document.
  • <html lang="en">: Root element with language attribute.
  • <head>: Metadata section (charset, viewport, links).
  • <meta charset="UTF-8">: Character encoding.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Responsive settings.
  • <link>: Links to external styles (CSS, Unicons, favicon).
  • <title>: Sets the page title.
  • <body>: Contains all visible content.
  • <div class="container">: Main wrapper div.

II. Navigation (Header/Nav)

  • <nav id="header">: Navigation bar.
  • <div class="nav-logo">: Logo and branding ("EMR RENOVATION").
  • <div class="nav-menu" id="myNavMenu">: Container for nav links.
  • <ul class="nav_menu_list">: Unordered list for navigation items.
  • <a href="#home" class="nav-link active-link">: Link with active class.
  • <div class="nav-button">: Button for downloading gallery (PDF).
  • <div class="nav-menu-btn">: Mobile menu button (bars icon).

III. Main Content

  • <main class="wrapper">: Main wrapper for content sections.
  • <section id="home">: Home/landing area.
  • <div class="featured-text">: Welcome text with typing effect.
  • Buttons: "Hire Us" (Instagram link) and "Download Gallery" (PDF).
  • <div class="social_icons">: Social media icons.
  • <div class="featured-image">: Main image showcase.
  • <a href="#about" class="scroll-btn">: Scroll down button to About section.
  • <section id="about">: About Us description.
  • <section id="projects">: Showcases project numbers, clients, and years of experience.
  • <section id="contact">: Contact form section (Name, Email, Message).

IV. Footer

  • <footer>: Footer wrapper element.
  • <div class="top-footer">: Top area ("EMR Renovation").
  • <ul class="footer-menu">: Footer navigation links.
  • <div class="footer-social-icons">: Social media links in footer.
  • <div class="bottom-footer">: Bottom copyright text.

V. Scripts

  • <script src="https://unpkg.com/typed.js@2.0.16/dist/typed.umd.js"></script>: Typing text animation.
  • <script src="https://unpkg.com/scrollreveal"></script>: Reveal animations on scroll.
  • <script src="scripts.js"></script>: Custom JavaScript logic.
