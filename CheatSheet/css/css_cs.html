I. Basics

  • @import url(...): Imports the Poppins font from Google Fonts.
  • :root: Defines CSS variables for easy theming.
  • *: Universal selector to reset margins and padding.
  • html: Sets scroll-behavior: smooth.
  • ::-webkit-scrollbar: Styles scrollbars for WebKit browsers.

II. Reusable Styles

  • .btn: Button styles (padding, background, etc.).
  • .section, .row, .col: Layout helpers (Flexbox).
  • .top-header: Section headers styling.

III. Navigation (nav)

  • nav: Main navbar styling.
  • .nav-logo, .nav-menu, .nav-link: Logo and menu styling.
  • .active-link::after: Indicator for active links.

IV. Featured Section (featured-box)

  • .featured-box: Layout for the main section.
  • .featured-text, .featured-name, .typedText: Text styles.
  • .featured-image: Styles the image and its animation.

V. About Section

  • .about-info, .about-btn: About box and button styles.
  • .skills-box, .skills-list: Skills display.

VI. Projects Section

  • .project-container, .project-box: Project display layout.
  • ::after, :hover: Hover effects.

VII. Contact Section

  • .contact-info, .form-control: Contact form styles.
  • .form-inputs, .input-field: Input fields and textarea.

VIII. Footer

  • footer, .top-footer, .middle-footer, .bottom-footer: Footer styles.

IX. Media Queries

  • @media (max-width: 1024px): Tablet styles.
  • @media (max-width: 900px): Mobile styles.
  • @media (max-width: 540px): Small devices styles.
